# Qusai Furniture
an attempt to build an API ecommerce website with #Laravel #5.8

It's still WIP. `NOT for production use.`

#### To build and test

```
php artisan migrate
```

#### Check routes:
```
php artisan route:list
```

#### To serve
```
php artisan serve
```

Then go to http://127.0.0.1:8000/api/product
